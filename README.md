# 📱 Scientific Calculator Android App

## 🚀 Project Overview
This is an Android-based Scientific Calculator application developed using Java in Android Studio.  
It supports both basic and advanced mathematical operations including scientific functions.

---

## ✨ Features
- Basic arithmetic operations (+, -, ×, ÷)
- Scientific functions (sin, cos, tan, log, etc.)
- Clean and simple UI
- Responsive button layout
- Easy-to-use calculator interface

---

## 🛠️ Tech Stack
- Language: Java
- Platform: Android Studio
- UI: XML Layout
- OS: Android

## ⚙️ Installation & Setup

### 1. Clone repository
git clone https://github.com/your-username/Android_Scientific_Calculator.git

### 2. Open project
Open Android Studio → Click "Open Project" → Select folder

### 3. Sync Gradle
Let Android Studio download dependencies automatically

### 4. Run project
Click Run ▶ button to start app on emulator or device

---

## 📌 How it Works
- User enters input using buttons
- Expressions are processed in Java logic
- Mathematical functions are calculated
- Result is displayed instantly on screen
